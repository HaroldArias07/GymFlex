# PROYECTO: VirtualGym

VISION:
  * Ser de los mejores aplicativos a nivel nacional que incentiven el hacer ejercicio de de manera remota. 
  
  * Ofrecer una gran variedad de servicios y promociones de manera virtual de nuestro aplicativo.
  
  * Querer satisfacer con todas las necesidades de nuestro usuarios.
  
SPRINT - 1:
	Que se realizó bien
	- Se establecieron tiempos de trabajo comodos para el desarrollo del Sprint
	- Se pudo ver claramente un avance significativo en cada Stand-up meeting
	Que se realizó mal
	- Pese a los tiempos establecidos no se tomo en cuenta la organización de cada uno de los integrantes y eso afecto al final.
	- Se trabajo de manera libre y no se tuvo un control del avance de cada tarea.
	Acciones a realizar
	- Organizar mejor los tiempos y horarios consultando a cada integrante para concretar un horario solido y eficiente.
